# 图书管理系统

图书管理系统的介绍！！

## 功能描述

将系统的整体功能按照人员分为4个部分，分别是学生功能、系统管理员、学生管理员和图书管理员等模块。将会按照这四个部分进行介绍。

用例描述据举例：

1. 用例编号：US 02

   用例名称：借阅图书

   参与者：学生

   入口条件：以学生身份登录图书管理系统

   事件流：
      - 通常情况：
        - 输入借阅图书的书号等信息
        - 在图书系统中查找相应的图书
        - 查看图书的借阅情况
        - 借阅成功
      - 例外情况：
        - 图书不存在，重新输入图书信息
        - 图书已经借完，推荐进入预约图书

   出口条件：图书借阅成功或者失败
